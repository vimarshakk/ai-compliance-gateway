# Funding

ACG is an open-source project. We are committed to keeping the core platform free and open.

## How We're Funded

- **Enterprise Licenses**: Paid enterprise features and support
- **Managed Cloud**: Hosted ACG at [app.acg.ai](https://app.acg.ai)
- **Professional Services**: Custom integrations and consulting
- **Sponsorships**: Community and corporate sponsorships

## Supporting ACG

If you find ACG useful, consider supporting the project:

### GitHub Sponsors

[Sponsor ACG on GitHub](https://github.com/sponsors/acg-dev)

### Corporate Sponsorship

Corporate sponsorships provide:
- Logo placement on our website and documentation
- Priority support and feature requests
- Early access to new features
- Custom integrations

Contact: sponsors@acg.dev

### Community Contributions

- **Code**: Contribute features, fix bugs, improve documentation
- **Issues**: Report bugs, request features
- **Discussions**: Help others in GitHub Discussions
- **Content**: Write blog posts, create tutorials, share use cases

## Open Core Model

| Compliance packs | ✅ | ✅ |
